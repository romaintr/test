Le Box Model

  1. Dans le fichier index.html ajouter une div avec une class .feature autour de chacun des blocs HTML qui présentent les principales features de l'application.

    
  
    Un défi chaque jour
    
  
    Recevez de nouveaux challenges tous les jours

  2. Dans le fichier style.css ajoutez le selecteur .feature et appliquez une bordure 

     .feature {border: 1px solid #F2F2F2;}

  3. Dans le même fichier style.css ajoutez le sélecteur #apropos qui correspond à la section qui entoure nos divs class="feature" Appliquez lui la régle margin: 0 auto; afin de centrer les divs à l'interieur. Définissez une largeur pour la section de 80% à l'aide de la propriété width 

     #apropos{
  margin: 0 auto;
  width: 80%;
}

  4. Nous allons ensuite définir les espacements. Sur le selecteur .feature ajoutez lui un margin un padding et centrez le texte avec la propriété text-align 

    margin: 15px;
padding: 15px 0;
text-align: center;
width: 400px;

  5. Pour finir vous pouvez appliquer une couleur de fond à votre div .feature en ajoutant la propriété background 

     background: #FAFAFA;

Alignement et positionnement

  1. La balise nav du fichier index.html contient un menu sous forme de liste qui s'affiche par défaut en display:block les uns en dessous des autres. Pour aligner le menu, dans le fichier style.css ajoutez le selecteur nav li en lui appliquant la propriété display:inline; et ajoutez un margin de 10px; 

    nav li {
  margin: 10px;
  display:inline;
}

  2. Afin d'aligner le logo et le menu sur la gauche ajoutez la propriété float:left; sur le selecteur nav ul et #logo 

    nav ul {
  float: left;
  margin-top: 25px;
}

#logo{
  color: #fff;
  font-size: 35px;
  font-family: 'Pacifico', cursive;
  float: left;
  margin-left: 50px;
}

  3. Ajoutez la propriété display:flex sur la section #apropos pour aligner les éléments contenus. Regardez la page et il y a un probleme tous les élements sont comprimés sur une meme ligne

  4. Pour que les élements puissent revenir à la ligne ajoutez la propriété flex-wrap: wrap; et ensuite centrez les avec la propriété justify-content: center;

  5. Pour finir nous allons appliquer un effet de déplacement au passage de la souris sur le lien d'action de la section #baseline. Il faut ajouter au CSS le selecteur #baseline a:hover avec les propriétés suivantes 
    #baseline a:hover {
  position: relative;
  top:1px;
}
